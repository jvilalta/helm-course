== Course Overview

This course teaches the fundamentals and practical usage of Helm for Kubernetes. Students will learn how to install, configure, customize, package, and manage Kubernetes applications using Helm charts.

The course is designed with a hands-on approach and includes practical exercises and real-world deployment patterns.

== Prerequisites

* Basic command-line familiarity
* Basic understanding of containers and Kubernetes
* Docker installed locally
* A code editor such as VSCode

== Learning Outcomes

By the end of this course, students will be able to:

* Install and configure Helm
* Deploy applications using Helm charts
* Customize deployments with values files
* Create and template custom charts
* Manage Helm release lifecycles
* Package and distribute charts
* Apply Helm best practices in real-world environments
